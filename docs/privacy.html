Privacy Policy

Data Collection

This extension:

  • Only reads the text content of web pages you choose to summarize
  • Sends this text to OpenRouter's API for summarization
  • Does not store or transmit any personal information
  • Does not track your browsing history

API Usage

  • We use OpenRouter's API to generate summaries
  • Only the text content of the current webpage is sent to the API
  • No personal data or browsing history is collected or transmitted

Data Storage

  • No user data is stored
  • Summaries are temporary and are cleared when you close the side panel
  • No cookies are used
  • No analytics are collected

Permissions

This extension requires permissions to:

  • Access the current tab's content (to read the text for summarization)
  • Connect to OpenRouter's API (to generate summaries)
  • Display ads through Google AdSense
